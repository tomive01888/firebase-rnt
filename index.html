Firebase Project Setup Guide

1. For you to setup a Firebase project, you're going to need a Google account.

2. Login to Firebase console.

3. Start up a new project and follow the step-by-step to create your first project.

Prepare VS Code and Firebase Initialization

1. To install Firebase, we will start by adding it as a dependency.

2. Run the following command:

npm install firebase

3. Now, go back to the Firebase Console.

4. Click the 'âš™ï¸ > Settings Icon > Project settings' and navigate to the 'General' tab.

5. Scroll down until you see your Firebase configuration code. Copy it and create a new file named firebaseInit.mjs in the root of your project.

6. Paste the copied Firebase configuration code into firebaseInit.mjs and ensure that you export the app instance.

- Example: export const app = initializeApp(firebaseConfig);

7. Import {getAuth} from 'firebase/auth' and create a new 'export const' called 'auth':

- Example:import { getAuth } from "firebase/auth";

- Example:export const auth = getAuth(app);

7. With this setup complete, your Firebase project is now ready to use! ðŸŽ‰

- !Notice: Remember to replace all "value" with your actual Firebase configuration values. 
                  
// firebaseInit.mjs
import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";
          
  const firebaseConfig = {
      apiKey: "value",
      authDomain: "value",
      projectId: "value",
      storageBucket: "value",
      messagingSenderId: "value",
      appId: "value"
  };
          
export const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);

Enabling Firebase authorization methods

In this setup I will show you the basic email and password authentication method.

1. Navigate to the Firebase console.

2. On left hand side you can see 3 dropdown menus, 'Build' , 'Run' and 'Analytics'.

3. Click 'Build' to expand and click 'Authentication'.

4. Once there, first tab will show a list of registered users, enter tab two 'Sign-in method'.

5. Now you can click the button that says 'Add new provider' and that should allow you to pick from many of the various options Firebase offer from different providers.
